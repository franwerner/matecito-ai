# Dominio: `privacy`

Cómo se trata la información personal con criterio de derechos: minimización de datos, consentimiento, anonimización/seudonimización, propósito de uso.

## Criterio de pertenencia

Un concern nuevo va en `privacy` si trata sobre *qué datos personales se tiene derecho a tener y con qué propósito*. Distinto de `security` (proteger los datos que ya se tienen) y de `compliance` (cumplir una regulación formal).
